## Learning radio interferometry with APSYNSIM

Install APSYNSIM and APSYNTRUE from
https://github.com/marti-vidal-i/APSYNSIM

The easiest is to clone the repository.

## Learning radio interferometry with Difmap

Install the software Difmap by following the instructions in 

[./difmap/README.md](./difmap/README.md)

The original software is in: 

https://ftp.eso.org/scisoft/scisoft4/sources/difmap/difmap.html

There, you can find also the Cookbook, but I've donwloaded it for convenience, and can be found in [./difmap/difmap-cookbook.ps](./difmap/difmap-cookbook.ps)

We'll use this software to obtain radio interferometry images of real source in the radio sky!
